# {frontmatter.title}

В этой документации вы узнаете, как установить и настроить плагин интеграции 1C:Enterprise для Medusa.

## Требования

- Medusa v2.14.0 или новее
- Node.js v20 или новее

---

## Установка

<CodeTabs>
  <CodeTab label="yarn" value="yarn">
    ```bash
    yarn add @gorgo/medusa-1c
    ```
  </CodeTab>
  <CodeTab label="npm" value="npm">
    ```bash
    npm install @gorgo/medusa-1c
    ```
  </CodeTab>
</CodeTabs>

---

## Настройка

Добавьте конфигурацию плагина в `medusa-config.js` вашего приложения Medusa:

```ts title="medusa-config.js"
# ...
module.exports = defineConfig({
  # ...
  plugins: [
    # ...
    {
      resolve: "@gorgo/medusa-1c",
      options: {}
    }
  ],
})
```

<Note>🚧 Интеграция в настоящее время находится в разработке, см. [Roadmap](./#roadmap).</Note>
