# CisStatusType

Статус проверки кода маркировки в системе :no-translate[«Честный ЗНАК»]:  * `FAILED` — не удалось проверить код.    Повторите попытку позже или удалите код маркировки.  * `IN_PROGRESS` — в процессе проверки.  * `NOT_ON_VALIDATION` — код маркировки не отправлен на проверку.  * `OK` — проверка успешно пройдена.  * `INVALID` — проверка не пройдена. Продажа товара с этим кодом запрещена. 

## Enum

* `Ok` (value: `'OK'`)

* `Failed` (value: `'FAILED'`)

* `InProgress` (value: `'IN_PROGRESS'`)

* `Invalid` (value: `'INVALID'`)

* `NotOnValidation` (value: `'NOT_ON_VALIDATION'`)
