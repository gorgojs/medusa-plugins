# FeedbackReactionStatusType

Статус реакции на отзыв:  * `ALL` — все отзывы.  * `NEED_REACTION` — отзывы, на которые нужно ответить. 

## Enum

* `All` (value: `'ALL'`)

* `NeedReaction` (value: `'NEED_REACTION'`)
