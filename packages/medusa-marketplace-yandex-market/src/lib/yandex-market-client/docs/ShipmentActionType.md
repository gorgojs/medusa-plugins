# ShipmentActionType

Действия с отгрузкой:  * `CONFIRM` — подтвердить отгрузку. * `DOWNLOAD_ACT` — скачать акт приема-передачи отгрузки. * `DOWNLOAD_INBOUND_ACT` — скачать список принятых заказов. * `DOWNLOAD_DISCREPANCY_ACT` — скачать акт расхождений. * `DOWNLOAD_TRANSPORTATION_WAYBILL` — скачать транспортную накладную. * `CHANGE_PALLETS_COUNT` — указать количество палет.
