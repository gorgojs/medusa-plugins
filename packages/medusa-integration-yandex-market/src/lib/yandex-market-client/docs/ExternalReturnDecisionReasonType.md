# ExternalReturnDecisionReasonType

Причины возврата:  * `BAD_QUALITY` — бракованный товар (есть недостатки).  * `DOES_NOT_FIT` — товар не подошел.  * `WRONG_ITEM` — привезли не тот товар. 

## Enum

* `BadQuality` (value: `'BAD_QUALITY'`)

* `DoesNotFit` (value: `'DOES_NOT_FIT'`)

* `WrongItem` (value: `'WRONG_ITEM'`)
