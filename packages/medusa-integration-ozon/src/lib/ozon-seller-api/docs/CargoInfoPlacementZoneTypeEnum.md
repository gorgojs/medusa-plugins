# CargoInfoPlacementZoneTypeEnum

Тип зоны размещения:  - `UNSPECIFIED` — не требуется распределение по зонам размещения;  - `UNDEFINED` — не определён;  - `SINGLE` — одинаковая зона размещения в составе грузоместа;  - `MULTI` — разная зона размещения в составе грузоместа. 

## Enum

* `Unspecified` (value: `'UNSPECIFIED'`)

* `Undefined` (value: `'UNDEFINED'`)

* `Single` (value: `'SINGLE'`)

* `Multi` (value: `'MULTI'`)
