# V1DraftMultiClusterCreateRequestDeleteSkuModeEnum

Режим удаления SKU, которые не попали в поставку.  Возможные значения:  - `PARTIAL` — система удалит только те единицы SKU, которые не прошли проверку;  - `FULL` — система удалит все единицы SKU, если хотя бы 1 единица этого SKU не прошла проверку. 

## Enum

* `Partial` (value: `'PARTIAL'`)

* `Full` (value: `'FULL'`)
