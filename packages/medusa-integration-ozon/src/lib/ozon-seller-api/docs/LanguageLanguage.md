# LanguageLanguage

Язык в ответе: - `EN` — английский, - `RU` — русский, - `TR` — турецкий, - `ZH_HANS` — китайский.  По умолчанию используется русский язык. 

## Enum

* `Default` (value: `'DEFAULT'`)

* `Ru` (value: `'RU'`)

* `En` (value: `'EN'`)

* `Tr` (value: `'TR'`)

* `ZhHans` (value: `'ZH_HANS'`)
