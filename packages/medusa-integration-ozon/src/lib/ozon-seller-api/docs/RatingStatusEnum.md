# RatingStatusEnum

Статус рейтинга: - `UNKNOWN` — не определён; - `OK` — хороший; - `WARNING` — показатели требуют внимания; - `CRITICAL` — критичный. 

## Enum

* `Unknown` (value: `'UNKNOWN'`)

* `Ok` (value: `'OK'`)

* `Warning` (value: `'WARNING'`)

* `Critical` (value: `'CRITICAL'`)
