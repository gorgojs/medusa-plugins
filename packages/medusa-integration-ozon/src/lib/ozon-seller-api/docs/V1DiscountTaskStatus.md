# V1DiscountTaskStatus

Статус заявки на скидку: - `NEW` — новая, - `SEEN` — просмотренная, - `APPROVED` — одобренная, - `PARTLY_APPROVED` — одобренная частично, - `DECLINED` — отклонённая, - `AUTO_DECLINED` — отклонена автоматически, - `DECLINED_BY_USER` — отклонена покупателем, - `COUPON` — скидка по купону, - `PURCHASED` — купленная.

* `Coupon` (value: `'COUPON'`)

* `Purchased` (value: `'PURCHASED'`)
