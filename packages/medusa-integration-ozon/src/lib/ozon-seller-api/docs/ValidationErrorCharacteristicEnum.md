# ValidationErrorCharacteristicEnum

Характеристика, по которой товар не прошёл проверку.    Возможные ограничения: - `UNSPECIFIED` — не определено; - `LENGTH` — длина, - `WIDTH` — ширина, - `HEIGHT` — высота, - `WEIGHT` — вес, - `SUM_OF_DIMENSIONS` — сумма измерений, - `VOLUME_WEIGHT` — объёмный вес, - `VOLUME` — объём, - `PRICE` — цена, - `LONGEST_SIDE` — самая длинная сторона.
