# SupplyOverdueReasonEnum

Причина просрочки поставки: - `UNSPECIFIED` — не определена; - `ORDER_TIMESLOT_EXPIRED` — поставка не доставлена в указанный таймслот; - `ORDER_TIMESLOT_NOT_SELECTED` — таймслот не указан вовремя; - `NOT_READY_FOR_PICKUP` — пикап-поставка не приведена в статус `ReadyToSupply` вовремя; - `PICKUP_FAILED` — курьер не смог забрать поставку; - `UNDEFINED` — неизвестная. 

## Enum

* `Unspecified` (value: `'UNSPECIFIED'`)

* `OrderTimeslotExpired` (value: `'ORDER_TIMESLOT_EXPIRED'`)

* `OrderTimeslotNotSelected` (value: `'ORDER_TIMESLOT_NOT_SELECTED'`)

* `NotReadyForPickup` (value: `'NOT_READY_FOR_PICKUP'`)

* `PickupFailed` (value: `'PICKUP_FAILED'`)

* `Undefined` (value: `'UNDEFINED'`)
