# SupplyOrderContentUpdateStatusResponseStatusEnum

Статус редактирования товарного состава поставки.  Возможные статусы: - `SUCCESS` — товарный состав изменён, - `IN_PROGRESS` — товарный состав в процессе изменения, - `ERROR` — возникла ошибка при изменении товарного состава. 

## Enum

* `Success` (value: `'SUCCESS'`)

* `InProgress` (value: `'IN_PROGRESS'`)

* `Error` (value: `'ERROR'`)
