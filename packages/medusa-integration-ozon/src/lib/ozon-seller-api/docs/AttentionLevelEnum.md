# AttentionLevelEnum

Доля влияния кластера на общий показатель:  - `UNKNOWN` — неизвестное значение;  - `LOW` — хороший показатель поставки товаров;  - `MEDIUM` — требуется поставка недостающих товаров;  - `HI` — требуется поставка недостающих товаров в первую очередь. 

## Enum

* `Unknown` (value: `'UNKNOWN'`)

* `Low` (value: `'LOW'`)

* `Medium` (value: `'MEDIUM'`)

* `Hi` (value: `'HI'`)
