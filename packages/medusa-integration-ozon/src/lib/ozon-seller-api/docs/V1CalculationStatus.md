# V1CalculationStatus

Статус создания черновика заявки на поставку: - `CALCULATION_STATUS_FAILED` — не удалось создать черновик, - `CALCULATION_STATUS_SUCCESS` — черновик создан, - `CALCULATION_STATUS_IN_PROGRESS` — черновик создаётся, - `CALCULATION_STATUS_EXPIRED` — истёк срок действия черновика. 

## Enum

* `CalculationStatusFailed` (value: `'CALCULATION_STATUS_FAILED'`)

* `CalculationStatusSuccess` (value: `'CALCULATION_STATUS_SUCCESS'`)

* `CalculationStatusInProgress` (value: `'CALCULATION_STATUS_IN_PROGRESS'`)

* `CalculationStatusExpired` (value: `'CALCULATION_STATUS_EXPIRED'`)
