# `@gorgo/medusa-payment-yookassa` example

Examples for the [@gorgo/medusa-payment-yookassa](https://www.npmjs.com/package/@gorgo/medusa-payment-yookassa) plugin.

## Prerequisites

- All the [common prerequisites](../README.md#prerequisites).
- A YooKassa account – [sign in or create one](https://yookassa.ru/joinups/?source=ks), a shop identifier `shopId` and a key `secretKey`.

## Configuration

Set up environment variables for [`medusa`](./medusa):

```
cd medusa
cp .env.template .env
# and configure your own `YOOKASSA_SHOP_ID` and `YOOKASSA_SECRET_KEY` inside .env
```
