# Расширение Admin UI для фида Яндекса

Плагин фида Яндекса предоставляет расширение в панели Medusa Admin, которое позволяет управлять конфигурациями фидов и отслеживать их статус.

## Доступ к разделу фидов

Откройте панель администратора Medusa и перейдите в раздел **Настройки** -> **Фиды** в меню Расширения.

![Страница списка фидов](https://github.com/user-attachments/assets/7998358a-a0ed-4f46-8927-0db3189aea31)

## Управление фидами

Интерфейс позволяет:

- Создавать новые конфигурации фидов
- Редактировать метаданные существующих фидов
- Настраивать расписания экспорта
- Выбирать категории товаров для экспорта
- Отслеживать статус генерации фидов
- Вручную запускать генерацию фидов
- Получать доступ к URL сгенерированных фидов

## Конфигурация фида

При создании или редактировании фида можно настроить:

- **Название фида**: Отображаемое имя фида
- **Имя файла**: Имя сгенерированного XML-файла
- **Данные магазина**: Имя, компания, URL и информация о платформе для вывода YML
- **Расписание экспорта**: Интервал для автоматической генерации фида
- **Выбор категории**: Какие категории товаров включить в фид
- **Сопоставление таксономии**: Как сопоставлять категории Medusa с категориями Яндекс.Маркет

## Мониторинг статуса фида

Интерфейс администратора показывает статус каждого фида, включая:

- Время последней генерации
- Статус генерации (успешно/ошибка)
- Следующую запланированную генерацию
- Детали ошибки, если генерация не удалась

## Доступ к сгенерированным фидам

Каждая конфигурация фида предоставляет прямой URL для доступа к сгенерированному XML-файлу:

```
http://{YOUR_MEDUSA_DOMAIN}/feeds/{ID}/{FILE_NAME}.xml
```

Этот URL можно использовать непосредственно в Яндекс.Маркет для импорта товаров.

## Ручная генерация

Используйте кнопку **Запустить сейчас**, чтобы вручную запустить генерацию фида при необходимости, например, когда вы добавили новые товары или обновили существующие и хотите обновить фид немедленно, не дожидаясь запланированной генерации.
