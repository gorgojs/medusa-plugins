# Справочник по API плагинов

Полный справочник по конечным точкам и методам API плагинов.

## Доступные конечные точки

### GET /api/plugins
Получить список доступных плагинов.

### POST /api/plugins/\{id\}
Активировать определенный плагин.

### DELETE /api/plugins/\{id\}
Деактивировать определенный плагин.

## Аутентификация

Все вызовы API требуют действительный токен аутентификации в заголовке Authorization.
