# {frontmatter.title}

A Medusa plugin that provides YooKassa payments.

## Plugin Features

- **Full Integration with YooKassa**  
  Connect your store to YooKassa for secure and reliable payments.
- **Receipt Generation**  
  Online receipts compliant with Federal Law No. 54.
- **One-Step & Two-Step Payments**  
  Autocapture or authorization/hold payment flows.
- **Refunds & Cancellations**  
  Full refund and order cancellation support.
- **Webhook Notifications**  
  Real-time payment status updates with signature verification.
- **Detailed Logging**  
  Request and response logs for debugging and support in development mode.
- **Storefront Example**  
  A complete <a href="https://github.com/sergkoudi/medusa-payment-yookassa/tree/main/examples/medusa-storefront" target="_blank">Next.js storefront</a> ready to use as a reference.

## What is YooKassa?

YooKassa is an online payment service by YooMoney that enables businesses to accept payments on their website or in an app. It supports various payment methods and ensures secure transaction processing. Learn more on the <a href="https://yookassa.ru" target="_blank">YooKassa website</a>.
