# Установка плагинов

Это руководство проведет вас через различные методы установки плагинов в вашей системе.

## Предварительные требования

Перед установкой плагинов убедитесь, что у вас есть:
- Работающий экземпляр основной системы
- Административные привилегии
- Совместимая версия основной системы (минимум v2.0)

## Методы установки

### Метод 1: Менеджер пакетов
Рекомендуемый подход для большинства пользователей:
1. Откройте терминал
2. Перейдите в каталог проекта
3. Выполните команду установки: `npm install plugin-name` или `yarn add plugin-name`

### Метод 2: Ручная установка
Для опытных пользователей, которым нужно больше контроля:
1. Скачайте пакет плагина
2. Извлеките в каталог плагинов
3. Обновите файлы конфигурации
4. Перезапустите систему

## Послестанционные шаги

1. Проверьте установку с помощью команды состояния
2. Настройте необходимые параметры
3. Проверьте базовую функциональность
4. Включите плагин в ваших настройках

## Проверка

После установки проверьте, что плагин работает правильно, запустив тестовые команды или проверив список плагинов.

Для получения подробных инструкций по каждому методу установки см. конкретные руководства в этом разделе.
