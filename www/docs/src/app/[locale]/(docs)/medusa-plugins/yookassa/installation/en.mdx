# Plugin Installation

This guide will walk you through the various methods to install plugins in your system.

## Prerequisites

Before installing plugins, ensure you have:
- A running instance of the core system
- Administrative privileges
- Compatible version of the core system (minimum v2.0)

## Installation Methods

### Method 1: Package Manager
The recommended approach for most users:
1. Open your terminal
2. Navigate to your project directory
3. Run the installation command: `npm install plugin-name` or `yarn add plugin-name`

### Method 2: Manual Installation
For advanced users who need more control:
1. Download the plugin package
2. Extract to the plugins directory
3. Update configuration files
4. Restart the system

## Post-Installation Steps

1. Verify installation with the status command
2. Configure necessary settings
3. Test basic functionality
4. Enable the plugin in your settings

## Verification

After installation, verify the plugin is working correctly by running test commands or checking the plugin list.

For detailed instructions on each installation method, see the specific guides in this section.
