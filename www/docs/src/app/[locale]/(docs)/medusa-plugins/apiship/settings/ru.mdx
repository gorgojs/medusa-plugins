# {frontmatter.title}

В этом руководстве вы узнаете, как управлять данными интеграции ApiShip и её подключениями.

## Просмотр данных провайдера ApiShip

Чтобы открыть страницу с настройками провайдера ApiShip:

1. Откройте Medusa Admin.
2. Перейдите в **Настройки → ApiShip**.

Откроется страница настроек ApiShip, где можно просматривать основные данные провайдера, информацию об отправителе, параметры товаров по умолчанию, настройки оплаты и список подключений.

![Общий вид настроек ApiShip](/static/apiship/ru/settings-overview.png)

* * *

## Редактирование данных провайдера ApiShip

В первой секции страницы можно изменить API-токен и режим работы провайдера.

Чтобы отредактировать данные провайдера:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **ApiShip** нажмите на меню действий в заголовке секции.
3. Выберите **Редактировать**.
4. В открывшейся боковой панели измените API-токен или переключите режим работы.
5. Нажмите **Сохранить**.

![Редактирование данных провайдера ApiShip](/static/apiship/ru/edit-provider-details.png)

* * *

## Редактирование адреса отправителя

Секция **Отправитель** хранит данные отправителя по умолчанию, которые используются при создании заказа в ApiShip.

Если у выбранного склада заполнен адрес, плагин использует данные склада в заказе. Если части данных не хватает, значения из этой секции используются как резервные.

Чтобы отредактировать данные отправителя:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **Отправитель** нажмите на меню действий в заголовке секции.
3. Выберите **Редактировать**.
4. В открывшейся боковой панели обновите страну, адрес, ФИО или телефон.
5. Нажмите **Сохранить**.

![Редактирование адреса отправителя](/static/apiship/ru/edit-default-store-address.png)

* * *

## Редактирование оплаты и НДС

Секция **Оплата и НДС** управляет тем, как плагин заполняет платежные данные при создании заказов в ApiShip.

Если включен наложенный платеж, плагин передает в ApiShip сумму наложенного платежа. Ставка НДС для доставки также передается при создании заказа, когда включен наложенный платеж.

Чтобы изменить настройки оплаты и НДС:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **Оплата и НДС** нажмите на меню действий в заголовке секции.
3. Выберите **Редактировать**.
4. В открывшейся боковой панели измените настройку наложенного платежа или ставку НДС для доставки.
5. Нажмите **Сохранить**.

![Редактирование оплаты и НДС](/static/apiship/ru/edit-payment-and-tax.png)

* * *

## Редактирование параметров товаров по умолчанию

Секция **Параметры товаров по умолчанию** хранит резервные габариты и вес товаров.

Эти значения используются при создании заказа в ApiShip, если у варианта товара не заданы собственные длина, ширина, высота или вес.

Чтобы изменить параметры товаров по умолчанию:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **Параметры товаров по умолчанию** нажмите на меню действий в заголовке секции.
3. Выберите **Редактировать**.
4. В открывшейся боковой панели измените длину, ширину, высоту и вес по умолчанию.
5. Нажмите **Сохранить**.

![Редактирование параметров товаров по умолчанию](/static/apiship/ru/edit-default-product-sizes.png)

## Управление подключениями

Подключения связывают службу доставки в ApiShip с конкретным подключением аккаунта ApiShip. При создании заказа плагин использует активное подключение, которое соответствует выбранной службе доставки.

При необходимости к подключению также можно привязать пункт приема.

### Добавление подключения

Чтобы добавить подключение:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **Подключения** нажмите **Добавить**.
3. В открывшемся модальном окне:
   - Выберите подключение аккаунта.
   - При необходимости выберите пункт приема.
   - Укажите, должно ли подключение быть активным сразу.
4. Нажмите **Сохранить**.

![Добавление подключения ApiShip](/static/apiship/ru/add-connection.png)

### Редактирование подключения

Чтобы изменить подключение:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **Подключения** найдите нужное подключение.
3. Откройте меню действий у строки.
4. Выберите **Редактировать**.
5. В открывшейся боковой панели измените статус активности подключения.
6. Нажмите **Сохранить**.

![Редактирование подключения ApiShip](/static/apiship/ru/edit-connection.png)

### Удаление подключения

Чтобы удалить подключение:

1. Перейдите в **Настройки → ApiShip**.
2. В секции **Подключения** найдите нужное подключение.
3. Откройте меню действий у строки.
4. Выберите **Удалить**.
5. Подтвердите удаление.
