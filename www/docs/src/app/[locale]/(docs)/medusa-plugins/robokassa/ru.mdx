# {frontmatter.title}

Плагин Medusa для приёма платежей через Robokassa.

## Возможности плагина

- **Полная интеграция с Robokassa**  
  Подключение к платёжной системе Robokassa для надёжной обработки платежей.
- **Формирование чеков**  
  Онлайн-чеки в соответствии с 54-ФЗ.
- **Одно- и двухстадийная оплата**  
  Автосписание или авторизация с холдированием.
- **Возвраты и отмены**  
  Возврат средств и отмена заказов.
- **Вебхук-уведомления**  
  Статусы платежей в реальном времени с проверкой подписи.
- **Тестовый режим**  
  Имитация оплаты без реальных списаний.
- **Подробное логирование**  
  Логи запросов и ответов для отладки и поддержки в режиме разработки.
- **Пример витрины магазина**  
  Готовый <a href="https://github.com/gorgojs/medusa-plugins/tree/main/examples/payment-robokassa/medusa-storefront" target="_blank">пример витрины на Next.js</a> для использования как референс.

## Что такое Robokassa?

<a href="https://robokassa.com" target="_blank">Robokassa</a> — российский платёжный агрегатор, поддерживающий банковские карты, электронные кошельки, мобильные платежи, СБП и другие способы оплаты. Широко используется малым и средним бизнесом в России и Казахстана благодаря простой интеграции и широкому охвату платёжных методов.
