# Конфигурация плагинов

Узнайте, как настроить различные плагины для ваших конкретных нужд.

## Переменные окружения

- API_KEY: Установите здесь свой API-ключ
- DEBUG_MODE: Включите или отключите отладку

## Пользовательские настройки

Каждый плагин может иметь определенные параметры конфигурации, которые можно задать в файле конфигурации.
